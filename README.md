Практическая работа №4: Тестирование "Белым ящиком"
Разработка WPF-приложения для расчета математических функций с тестированием методом "белого ящика"

## Цель работы
Приобретение практических навыков ручного тестирования методом "белого ящика" и разработки WPF-приложений с многостраничным интерфейсом.
