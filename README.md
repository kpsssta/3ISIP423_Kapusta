## Выполнение работы

### 1. Создаём проект Bank

Был создан проект консольного приложения `Bank` с классом `BankAccount`, реализующим основные операции банковского счета
- `Debit` - снятие средств со счета
- `Credit` - внесение средств на счет

### 2. Документирование кода

В код были добавлены XML-комментарии для всех методов, свойств и констант
- `<summary>` - общее описание методов и классов
- `<param>` - описание параметров методов
- `<returns>` - описание возвращаемых значений
- `<exception>` - описание возможных исключений
- `<value>` - описание свойств

### 3. Создание тестового проекта

Был создан проект модульного теста `BankTests` с классом `BankAccountTests`
