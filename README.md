## Использованные методы Assert

| Метод | Назначение | Пример использования |
|-------|------------|---------------------|
| `Assert.AreEqual()` | Проверка равенства двух значений с заданной точностью | Проверка числовых результатов функций 1 и 3 |
| `Assert.IsTrue()` | Проверка истинности логического выражения | Проверка, что результат > 0 или не является NaN |



Тесты выполняют: 

TestFunction1_CalculateW - проверяет работает ли формула (т.е. подставляет разные значения) 
TestFunction2_CalculateD - работают ли все три ветки условия ( x>y, x<y, x=y с разными f(x))
TestFunction3_CalculateY - работает ли формула ( при x=0 → y=1 и т.д.)

## Вывод о проведённом тестировании

В ходе выполнения практической работы были разработаны три модульных теста для проверки корректности работы математических функций из 4-й практической работы.


**Причина успешного выполнения тестов:**
1. Математические функции реализованы в соответствии с заданными формулами без логических ошибок.
2. Входные данные для тестов подобраны корректно и покрывают различные случаи: нулевые значения, положительные значения, пограничные ситуации.
3. При сравнении чисел с плавающей точкой использована допустимая погрешность (1e-6), что позволяет избежать ложных ошибок из-за округления.
4. Использование методов `Assert.AreEqual` и `Assert.IsTrue` позволило проверить как точные равенства, так и условия (например, что результат больше нуля).
